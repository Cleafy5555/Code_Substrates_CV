# Substrates analysis

This repository contains cleaned and annotated R scripts for the paper "RECYCLED AND WASTE-DERIVED SUBSTRATES ENHANCE YIELD AND VEGETATIVE GROWTH BUT NOT CLUBROOT DISEASE OUTCOMES IN BROCCOLINI"



## Scripts

1. `01_polytunnel_biomass_screen.R`  
   Broad substrate screening for root, shoot, and leaf dry biomass.

2. `02_stem_nutrient_analysis.R`  
   Stem nutrient summaries and heatmap relative to control.

3. `03_floret_yield_timeseries_screen.R`  
   Time-series yield and floret count analyses for the screening experiment.

4. `04_maydec_biomass_dsi_inoculation.R`  
   May-Dec inoculation experiment: root/shoot dry weight and DSI.

5. `05_maydec_floret_quality_lmm.R`  
   Mixed model for floret bunch weight using treatment, infection status, floret traits, and harvest timing.

## Notes

These scripts have been cleaned. They are intended to be understandable and reproducible.
